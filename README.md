# 貪食蛇 Snake Game

> 基於 C++ 和 SFML(Simple and Fast Multimedia Library）製作

<img src="cover.jpg" style="display:flex; justify-content:center;"></img>

## 使用說明

1.先`clone`下來
```git=
git clone https://github.com/viiccwen/Snake-SFML.git
```

2. 進入clone下來的資料夾
3. 進入`release` 資料夾
4. 點擊`Snake Game.exe` 檔案
5. Have Fun！
